## Step 0 · 骨架提取

**问题决定图的类型**：书的核心是两种范式的对比关系，用对照表 + 因果链。

### 核心二元结构

```
                    ┌──────────────────────────────┐
                    │       游戏的两种形态           │
                    └──────────────────────────────┘
                           │              │
              ┌────────────┘              └────────────┐
              ▼                                        ▼
       有限游戏（Finite）                    无限游戏（Infinite）
       ──────────────                      ──────────────
       目的：取胜，终结游戏                  目的：延续游戏
       边界：外部固定（时间/空间/人数）       边界：内部生成，随威胁调整
       规则：博弈前锁定，中途不变             规则：博弈中共识修改
       惊奇：威胁，力图消除                   惊奇：燃料，欢迎接纳
       身份：头衔（来自过去的胜利）            身份：名字（指向开放的未来）
       态度：严肃（遮蔽自由）                 态度：游戏性（承认自由）
       力量：权力（剧本性，可量化比较）        力量：力量/Strength（传奇性，无法封闭量化）
       宏观：社会（保护头衔与财产的结构）      宏观：文化（不断越轨与创新的过程）
       失败：死亡 / 出局 / 永生作为最终脚本    失败：静默（无人倾听，游戏中止）
```

### 关键因果链

```
有限游戏思维 →
  → 固化边界 → 积累头衔 → 追求权力 → 需要观众认可 → 社会形成
  → 遮蔽自由 → 剧本化行为 → 严肃性 → 惊奇威胁 → 控制欲

无限游戏思维 →
  → 调整边界 → 生成名字 → 展示力量 → 邀请更多参与 → 文化形成
  → 承认自由 → 传奇化行为 → 游戏性 → 惊奇欢迎 → 开放性
```

## Step 1 · 概念速览

### 核心概念（含边界说明）

| 概念 | 通俗解释 | 具体例子 | 边界模糊处 |
|------|---------|---------|-----------|
| 有限游戏 | 以取胜为目的、有边界和终点的游戏 | 足球赛、法庭诉讼、竞争上岗 | 有时候一场游戏既有终点又在更大无限游戏中 |
| 无限游戏 | 以延续为目的、不断调整规则的游戏 | 科学探索、文化、友谊、语言 | 并非无目的漫游，仍有内部约束 |
| 权力（Power） | 游戏结束后观众赋予的剧本性力量，来自过去的胜利 | 冠军头衔、CEO职位、军衔 | 与物理力量混淆——但洪水不能"打败"人 |
| 力量（Strength） | 传奇性的，来自开放行动，允许他人也发起行动 | 教师的影响力、文化人物的遗产 | 无法量化，容易被误以为软弱 |
| 头衔（Title） | 指向过去封闭游戏的符号，要求观众认可 | 博士、将军、MVP | 名字有时也变成头衔（凯撒、拿破仑） |
| 名字（Name） | 指向开放未来的具体性，允许惊奇 | 直呼其名而非职衔称呼某人 | 名字本身也可被剧本化 |
| 社会（Society） | 有限游戏的积累结构，保护既有头衔和财产 | 法律体系、资质认证、晋升制度 | 社会也包容某种程度的文化行为 |
| 文化（Culture） | 无限游戏的传统，通过越轨延续传统 | 科学传统、艺术史、哲学 | 文化偏离≠切断传统，而是携带传统前行 |
| 严肃性（Seriousness） | 遮蔽自己参与游戏的自愿性，认为自己的角色是必须的 | "我不得不这么做" "这就是规则" | 不是负面词——有限游戏需要它 |
| 剧本性（Theatrical） | 权力的表演性质，需要观众确认 | 仪式、制服、消费炫耀 | 与艺术表演不同——艺术可以是无限游戏 |
| 传奇性（Dramatic/Legendary） | 结果开放、允许惊奇的行动模式 | 即兴创作、真正的对话、探险 | 不是戏剧性（dramatic 也有夸张义）|

### 需要重点验证的边界

- ★ 「有限游戏中的赢，在无限游戏里是什么？」（嵌套关系，第7节）
- ★ 「惊奇如何同时终结有限游戏、延续无限游戏？」（第17节）
- ★ 「权力为什么是剧本性的，不是真实的？」（第27-29节）
