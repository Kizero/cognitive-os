# Step 3–4 · 结构可视化 + 可执行模型

---

## Step 3 · 结构可视化

### 完整对照结构图

```
┌─────────────────────────────────────────────────────────────────┐
│                    有限游戏 vs 无限游戏                           │
├───────────────┬─────────────────────┬───────────────────────────┤
│  维度         │  有限游戏            │  无限游戏                 │
├───────────────┼─────────────────────┼───────────────────────────┤
│ 目的          │ 取胜，终结游戏       │ 延续游戏，让更多人参与    │
│ 边界来源      │ 外部固定（时/空/人） │ 内部生成，随威胁调整      │
│ 规则          │ 博弈前固定，中途不变 │ 博弈中共识修改            │
│ 对待惊奇      │ 威胁，力图消除       │ 燃料，欢迎接纳            │
│ 身份符号      │ 头衔（指向过去）     │ 名字（指向未来）          │
│ 自我态度      │ 严肃（遮蔽自由）     │ 游戏性（承认自由）        │
│ 力量形式      │ 权力（剧本性）       │ 力量/Strength（传奇性）   │
│ 宏观表现      │ 社会（保护头衔）     │ 文化（越轨与传承）        │
│ 对死亡        │ 失败（被终结）       │ 内置于游戏，凡人继续      │
│ 对胜利        │ 游戏终点，确认头衔   │ 游戏中的一个瞬间          │
│ 表演性        │ 剧本性（theatrical） │ 传奇性（dramatic/genuine）│
└───────────────┴─────────────────────┴───────────────────────────┘
```

### 嵌套关系图

```
无限游戏（文化/生命）
  └── 包含多个有限游戏（职业、竞争、比赛）
        └── 每个有限游戏的"赢"是无限游戏中的一个时刻
        └── 把有限游戏的赢等同于无限游戏的目的 → 框架性错误
```

### 权力-力量对立结构

```
权力（Power）                          力量（Strength）
  ├─ 来源：过去封闭游戏的胜利            ├─ 来源：开放行动，允许他人也行动
  ├─ 衡量：相对比较（比X强Y%）           ├─ 衡量：无法封闭量化
  ├─ 时间指向：过去                      ├─ 时间指向：未来
  ├─ 依赖：观众的仪式性认可              ├─ 依赖：他人的参与和回应
  └─ 本质：剧本性（theatrical）          └─ 本质：传奇性（genuine）
```

### 社会-文化对立结构

```
社会（Society）                        文化（Culture）
  ├─ 游戏性质：保护既有有限游戏规则       ├─ 游戏性质：无限游戏的传统延续
  ├─ 对越轨：禁止（威胁头衔有效性）       ├─ 对越轨：欢迎（越轨是文化本质）
  ├─ 时间观：宿命/不朽（重复过去）        ├─ 时间观：历史/开放（携带过去前进）
  ├─ 财产：剧本性地确认过去胜利           ├─ 财富：被消费而非积累（流动）
  └─ 例子：法律体系、资质认证、晋升制度   └─ 例子：科学传统、艺术史、哲学
```

## Step 4 · 可执行模型

**遇到X，做Y，因为Z**

---

### 规则1：识别游戏框架

**X**：面对任何竞争/冲突/选择情境，需要决策。

**Y**：先问三个问题，再制定策略：
1. 参与者的目的是终结（取胜）还是延续（持续）？
2. 边界是被视为固定障碍，还是可协商的工具？
3. 如果赢了，这个游戏结束了，我还想要什么？

**Z**：因为不识别游戏框架就制定策略，等于在错误的游戏里赢了——可能以终结无限游戏为代价赢得有限游戏。

---

### 规则2：检测"有限化"陷阱

**X**：一个组织/个人声称有"使命/愿景"（无限游戏语言），但行为上持续关注竞争对手动态、头衔积累、市场份额。

**Y**：观察行动，不看口号：
- 规则设计是保护现有玩家还是欢迎新玩家？
- 对惊奇/意外的反应是焦虑消除还是好奇探索？
- 成功指标是封闭可比的（份额、排名）还是开放生成的（影响范围、参与深度）？

**Z**：因为口说无限、心打有限是最常见的框架误用，它让组织用无限游戏的代价（不设防、开放）去打有限游戏（结果更易受伤）。

---

### 规则3：有限游戏的嵌套使用

**X**：必须参与一场明确的有限游戏（法律诉讼、标准竞标、晋升考核）。

**Y**：以无限游戏参与者的态度参与有限游戏：
- 全力认真参与（有限游戏需要适当的严肃性）
- 同时保持对嵌套关系的意识：这场有限游戏的结果服务于哪个无限游戏？
- 赢了之后问：我的无限游戏是否继续了，还是我把有限游戏的赢当成了终点？

**Z**：因为无限游戏可以包含有限游戏，但不反过来。有限游戏的赢只是无限游戏中的一个瞬间。

---

### 规则4：应对权力的剧本性

**X**：遇到权力展示（头衔、财产、制度权威）试图终止讨论或行动。

**Y**：
- 识别：这种权力是否需要我的认可才能生效？
- 如果是：我可以选择不进入这个有限游戏的剧本
- 如果不是（强制力）：评估进入有限游戏的代价，再决定是否参与

**Z**：因为权力是剧本性的，它依赖观众（包括被施加权力者）的认可才能生效。不认可不等于能免受代价，但识别这个结构可以让你在选择时更清醒。

---

### 规则5：文化越轨的判断

**X**：面对"这不符合规范/传统"的反对声音。

**Y**：区分两种越轨：
- 断裂型越轨：切断与过去的联系，导致传统遗忘 → 文化价值有限
- 携带型越轨：把过去已经开始的东西带向新的开放性 → 这是文化的本质

**Z**：因为文化不是偏离传统，而是传统本身通过越轨延续；真正的创新让人对熟悉事物产生陌生感，而不是彻底遗弃它。
